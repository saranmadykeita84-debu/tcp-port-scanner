## Utilisation 
Dans notre programme on a utiliser "visual code" pour réaliser le travail.
Pour pouvoir démarrez notre code il faut ouvrir le terminal dans "visual code" et on tape: python scanner.py < le nom de domaine ou son adresse IP>.
Pour que le programme fonctionne, on devrait l'ouvrir depuis don emplacement 


# Multithreading
Concernant le multithreading on l'a bien utiliser dans le programme, par exemple  quand on scanne les port réserver de 0 à 1023 ça nous renvois directement les ports ouverts sans plus tardé. Car sans le multithreading le programme serait obliger d'exécuté les ports un par un.
